# Gearman Example

Installation:

> sudo apt-get install gearman php-gearman gearman-tools

Run the example:

> \# start gearman server
>
> `sudo gearmand -d`
>
> \# run worker
>
> `php worker.php`
>
> \# run client
>
> `php client.php`
>

More details:

[!https://www.php.net/manual/en/book.gearman.php](https://www.php.net/manual/en/book.gearman.php)
